The KR260 Robotics dev-kit is comprised of a KR260 carrier board and a K26 SOM. The latter uses the K26 FPGA which corresponds to the ZU5EV FPGA, with the closest match part being the `xczu5ev-sfvc784-2LV-e` (which has a higher max temperature rating).

This device conttains the following logic resources:

| Resource | Quantity | z7020 |
|----------|----------|-------|
| LUTs     | 117K     | 53K   |
| FFs      | 234K     | 106K  |
| RAMB36s  | 144      | 140   |
| URAM288s | 64       | 0     |
| DSP      | 1248     | 220   |
